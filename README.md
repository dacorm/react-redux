# Проект: React-Redux
----------------------------------

Моя собственная реализация библиотек React и Redux

**Используемые технологии**
1. TypeScript
2. WebPack

Планы по доработке:
1. Доделать редакс

Инструкция по запуску:
1. Скачать папку с кодом
2. Открыть, запустить команду npm install
3. Запустить команду npm run dev
